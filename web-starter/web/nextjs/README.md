# Introduction

A simple Noir circuit with browser proving with bb.js
This is a Next.js version, similar to the vite and webpack examples.

Tested with Noir 1.0.0-beta.11, bb 0.87.0, and Next.js 15.

## Setup

```bash
(cd ../../circuits && ./build.sh)
yarn
```

## Run

```bash
yarn build
yarn start
```
